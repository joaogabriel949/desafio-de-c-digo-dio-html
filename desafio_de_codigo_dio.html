História dos computadores

Pelo que venho estudando, a história da computação é basicamente a evolução de como processamos dados. Começou com máquinas mecânicas e lógica puramente matemática até chegarmos nos processadores modernos. Achei muito interessante notar que os primeiros computadores ocupavam salas inteiras, e hoje rodamos sistemas complexos em máquinas super compactas. A evolução do hardware andou lado a lado com a criação da internet e das linguagens de programação. Para aprofundar, gosto de consultar materiais na Wikipedia sobre a história do PC.

O que são Clients?

Descobri que o Client (ou cliente) sou eu, ou melhor, é o meu navegador ou o aplicativo que estou usando no meu computador ou celular. Quando abro um site, minha máquina está atuando como cliente. O papel fundamental do cliente é fazer requisições para a rede e exibir a interface para nós, usuários, de forma amigável.

O que são Servers?

Se o cliente é quem pede, o Server (servidor) é a máquina que entrega. São computadores robustos, conectados o tempo todo, projetados exclusivamente para receber as requisições dos clientes, processar dados (como salvar as informações de um aplicativo de dieta no banco de dados) e devolver uma resposta. Sem os servers, a comunicação web não existiria.

Como é uma estrutura básica de um HTML?

Achei a estrutura do HTML muito intuitiva! Ela funciona como o esqueleto do site. Começamos avisando o navegador que é um documento moderno com a tag <!DOCTYPE html>. Em seguida, a tag <html> abraça todo o projeto. A página se divide em duas partes principais: o <head>, que é o "cérebro" (onde ficam configurações, título e coisas que o usuário não vê diretamente), e o <body>, que é o "corpo" da página, onde colocamos todo o conteúdo visível, como textos e imagens.

Entendendo a diferença entre listas ordenadas e não ordenadas

Essa parte foi bem tranquila de pegar. Usamos listas para organizar informações em tópicos, e a escolha da tag certa depende se a ordem das coisas importa ou não.

A lista não ordenada (usando a tag <ul>) serve para agrupar itens aleatórios, criando apenas "bolinhas" (bullets). Por exemplo, alguns jogos que curto jogar no tempo livre:

  • Valorant
  • Warzone
  • Counter-Strike 2

Já a lista ordenada (com a tag <ol>) é usada quando a sequência é fundamental e numerada. Por exemplo, os passos que segui para fazer o upgrade do meu setup:

  1. Adquirir o processador Ryzen 5 5600X.
  2. Comprar a placa de vídeo RX 7600.
  3. Instalar os componentes na placa-mãe.
